# Fullness Stationery データ管理サービス APIリファレンス

## 総合開発演習の解答例
管理者向けデータ管理サービス（UC009〜UC018）のクラス・インターフェイスのリファレンスです。

## 構成

本システムは4層で構成されています。

- **ドメイン層（Backend.Domain）** — 業務ルールとエンティティ（集約）
- **アプリケーション層（Backend.Application）** — ユースケースの実装
- **インフラストラクチャ層（Backend.Infrastructure）** — 永続化・外部サービス連携
- **プレゼンテーション層（Backend.Api）** — REST API のコントローラ

左上の「Api Documentation」または検索ボックスから、各クラスの詳細を参照できます。
